# 通过 OLED 显示三维图形

# 1. 硬件方案
1. 使用 STM32F103C8T6
2. 使用 IIC OLED(SSD1306) 
3. 使用电脑通过串口控制STM32
##### 引脚连接方式：
- PB7 --- SDA
- PB6 --- SCL
- PA3 --- TXD
- PA2 --- RXD

# 2. 使用方法
将串口连接到电脑，打开串口后写入:
### 1. 移动
- w 摄像机前进
- s 摄像机后退
- a 摄像机向左移动
- d 摄像机向右移动
- j 摄像机向下移动
- k 摄像机向上移动
### 2. 旋转
- q 摄像机逆时针旋转（从上往下看）
- e 摄像机顺时针旋转（从上往下看）
- z 摄像机抬头
- c 摄像机低头
- u 摄像机逆时针旋转（向左歪头）
- i 摄像机顺时针旋转（向右歪头）

### 2. 其他功能
- r 摄像机围绕 定义在main.cpp中的 void HAL_UART_RxCpltCallback(UART_HandleTypeDef *huart) 回调函数内的局部变量 pos_cube 点水平旋转
